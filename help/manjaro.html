# Manjaro Mirror

# Introduction

Manjaro (opens new window) is a easy-to-install rolling-release Linux distribution based on Arch Linux (opens new window).

# Configuration

Write the following line to /etc/pacman.d/mirrorlist:

Server = https://mirrors.sustech.edu.cn/manjaro/stable/$repo/$arch

# Update Repository Indexes

sudo pacman -Syy
